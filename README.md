# CuraPharm - Pharmacy Management Web Application

CuraPharm is a dynamic, multi-user Java Web Application designed to streamline pharmacy operations, manage medicine inventories, handle prescriptions, and process billing. 

## 🚀 Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap (Optional)
* **Backend:** Java Servlets, JSP (JavaServer Pages)
* **Server:** Apache Tomcat (v9.0 or v10.0)
* **Database:** MySQL
* **Build Tool/IDE:** Eclipse / IntelliJ IDEA

---

## 🛠️ Prerequisites & Local Setup

Before running this project locally, ensure you have the following installed:
1. **Java JDK 8 or higher**
2. **Apache Tomcat Server**
3. **MySQL Server** & MySQL Workbench
